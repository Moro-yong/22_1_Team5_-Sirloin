## 담당 역할
**상품 기본 정보 등록란**  
- 카테고리 데이터를 전달받아 리스팅 및 checkbox 클릭 시 해당 카테고리를 선택 리스트 생성
- 필터 태그 input value의 글자에 근접한 데이터 제공 및 태그 선택 시 리스트 생성
- 상품명 입력시 상품코드 생성 함수를 작성
- 상품 소개 value를 state에 저장
- 이미지 첨부 공통 컴포넌트로 작성
    - 썸네일 이미지는 1개의 이미지만 선택하도록 하였습니다.
    - 상품 대표 이미지는 여러개의 이미지를 선택 할수 있도록 하였습니다.
    - 파일 선택 후 이미지 재첨부하면 기존 이미지는 제거되도록 하였습니다.
    - 첨부한 이미지 파일명 노출 및 삭제 기능
